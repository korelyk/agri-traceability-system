## 第6页：相关技术 - 区块链

### 区块链核心特性

```
┌─────────────────────────────────────┐
│  去中心化  │  不可篡改  │  可追溯   │
└─────────────────────────────────────┘
         │           │           │
         ▼           ▼           ▼
    无中心节点    数据不可改    全程可追溯
    共同维护      历史可查      来源可溯
```

### 区块链数据结构

- **区块头**：前一区块哈希、Merkle根、时间戳、难度值、随机数
- **区块体**：交易列表
- **链式结构**：通过哈希值链接，形成不可篡改的链条

## 第10页：区块链核心设计

### 区块结构

```java
public class Block {
    // 区块头
    private String blockHash;        // 当前区块哈希
    private String previousHash;     // 前一区块哈希
    private String merkleRoot;       // Merkle树根哈希
    private long timestamp;          // 时间戳
    private int nonce;               // 随机数
    private int difficulty;          // 难度值
    
    // 区块体
    private List<Transaction> transactions;  // 交易列表
}
```

## 第11页：核心算法实现

### SHA-256哈希计算

```java
public String calculateHash() {
    String data = index + previousHash + 
                  merkleRoot + timestamp + nonce;
    MessageDigest digest = 
        MessageDigest.getInstance("SHA-256");
    byte[] hash = digest.digest(data.getBytes());
    // 转换为16进制字符串
    return bytesToHex(hash);
}
```

### 工作量证明挖矿

```java
public void mineBlock() {
    String target = new String(
        new char[difficulty]).replace('\0', '0');
    while (!blockHash.substring(0, difficulty)
           .equals(target)) {
        nonce++;
        blockHash = calculateHash();
    }
}
```

## 第12页：数字签名实现

### RSA密钥生成

```java
public static Map<String, String> generateKeyPair() {
    KeyPairGenerator keyGen = 
        KeyPairGenerator.getInstance("RSA");
    keyGen.initialize(2048);
    KeyPair keyPair = keyGen.generateKeyPair();
    
    // 编码为Base64
    String publicKey = Base64.encode(
        keyPair.getPublic().getEncoded());
    String privateKey = Base64.encode(
        keyPair.getPrivate().getEncoded());
    
    return Map.of("publicKey", publicKey, 
                  "privateKey", privateKey);
}
```

### 签名与验证

```java
// 签名
Signature sig = Signature.getInstance("SHA256withRSA");
sig.initSign(privateKey);
sig.update(data.getBytes());
byte[] signature = sig.sign();

// 验证
sig.initVerify(publicKey);
sig.update(data.getBytes());
boolean valid = sig.verify(signature);
```

## 第15页：系统特色与创新

### 系统特色

1. **数据不可篡改**
   - 区块链技术保证数据一旦上链无法修改
   
2. **身份可验证**
   - 数字签名确保每条记录来源可靠
   
3. **全程可追溯**
   - 从生产到销售的全生命周期记录
   
4. **防伪能力强**
   - 二维码+区块链双重防伪
   
5. **用户体验好**
   - 简洁直观的Web界面，支持扫码查询

- 轻量级区块链实现
