## 六、核心知识点

### 6.1 区块链核心概念

#### 区块结构

```
┌─────────────────────────────────────┐
│              区块头                  │
├─────────────────────────────────────┤
│  前一区块哈希  │  链接前一个区块      │
│  Merkle根     │  交易列表的哈希      │
│  时间戳       │  区块创建时间        │
│  难度值       │  挖矿难度目标        │
│  随机数       │  工作量证明          │
├─────────────────────────────────────┤
│              区块体                  │
├─────────────────────────────────────┤
│  交易列表     │  包含多条交易        │
└─────────────────────────────────────┘
```

#### 区块链不可篡改原理

1. 每个区块包含前一区块的哈希值
2. 修改任一区块的数据会改变其哈希值
3. 后续区块的前一哈希值将不匹配
4. 需要重新计算所有后续区块的哈希
5. 在算力有限的情况下几乎不可能实现

### 6.2 数字签名原理

```
发送方：                    接收方：
┌─────────┐               ┌─────────┐
│  数据    │               │  数据    │
└────┬────┘               └────┬────┘
     │                         │
     ▼                         ▼
┌─────────┐               ┌─────────┐
│ SHA-256 │               │ SHA-256 │
│  哈希   │               │  哈希   │
└────┬────┘               └────┬────┘
     │                         │
     ▼                         ▼
┌─────────┐    签名    ┌─────────┐
│ 私钥加密 │  ──────▶  │ 公钥解密 │
│  签名   │           │  验证   │
└─────────┘           └────┬────┘
                           │
                           ▼
                      ┌─────────┐
                      │ 哈希比较 │
                      │ 验证结果 │
                      └─────────┘
```

### 6.3 关键代码解读

#### 区块哈希计算

```java
public String calculateHash() {
    try {
        // 1. 拼接区块数据
        String data = index + previousHash + merkleRoot + 
                      timestamp + nonce + difficulty;
        
        // 2. 使用SHA-256计算哈希
        MessageDigest digest = MessageDigest.getInstance("SHA-256");
        byte[] hash = digest.digest(data.getBytes("UTF-8"));
        
        // 3. 转换为16进制字符串
        StringBuilder hexString = new StringBuilder();
        for (byte b : hash) {
            String hex = Integer.toHexString(0xff & b);
            if (hex.length() == 1) hexString.append('0');
            hexString.append(hex);
        }
        return hexString.toString();
    } catch (Exception e) {
        throw new RuntimeException("计算哈希失败", e);
    }
}
```

#### Merkle树计算

```java
public String calculateMerkleRoot() {
    if (transactions == null || transactions.isEmpty()) {
        return "";
    }
    
    // 1. 获取所有交易的哈希值
    List<String> treeLayer = new ArrayList<>();
    for (Transaction tx : transactions) {
        treeLayer.add(tx.getTransactionHash());
    }
    
    // 2. 逐层计算父节点哈希
    while (treeLayer.size() > 1) {
        List<String> newLayer = new ArrayList<>();
        for (int i = 0; i < treeLayer.size(); i += 2) {
            String left = treeLayer.get(i);
            String right = (i + 1 < treeLayer.size()) ? 
                           treeLayer.get(i + 1) : left;
            newLayer.add(applySha256(left + right));
        }
        treeLayer = newLayer;
    }
    
    // 3. 返回根哈希
    return treeLayer.get(0);
}
```

#### 工作量证明挖矿

```java
public void mineBlock() {
    // 1. 根据难度值生成目标字符串
    // 难度为4时，target = "0000"
    String target = new String(new char[difficulty])
                        .replace('\0', '0');
    
    // 2. 循环计算哈希，直到满足难度要求
    while (!blockHash.substring(0, difficulty).equals(target)) {
        nonce++;  // 递增随机数
        blockHash = calculateHash();  // 重新计算哈希
    }
    
    System.out.println("区块挖矿成功: " + blockHash);
}
```

#### RSA数字签名

```java
// 签名过程
public String sign(String data, String privateKeyBase64) {
    // 1. 解码私钥
    byte[] privateKeyBytes = Base64.getDecoder()
                            .decode(privateKeyBase64);
    PKCS8EncodedKeySpec keySpec = 
        new PKCS8EncodedKeySpec(privateKeyBytes);
    KeyFactory keyFactory = KeyFactory.getInstance("RSA");
    PrivateKey privateKey = keyFactory.generatePrivate(keySpec);
    
    // 2. 创建签名对象
    Signature signature = Signature.getInstance("SHA256withRSA");
    signature.initSign(privateKey);
    signature.update(data.getBytes(StandardCharsets.UTF_8));
    
    // 3. 生成签名
    byte[] signatureBytes = signature.sign();
    return Base64.getEncoder().encodeToString(signatureBytes);
}

// 验证过程
public boolean verify(String data, String signatureBase64, 
                      String publicKeyBase64) {
    // 1. 解码公钥
    byte[] publicKeyBytes = Base64.getDecoder()
                           .decode(publicKeyBase64);
    X509EncodedKeySpec keySpec = 
        new X509EncodedKeySpec(publicKeyBytes);
    KeyFactory keyFactory = KeyFactory.getInstance("RSA");
    PublicKey publicKey = keyFactory.generatePublic(keySpec);
    
    // 2. 解码签名
    byte[] signatureBytes = Base64.getDecoder()
                           .decode(signatureBase64);
    
    // 3. 验证签名
    Signature signature = Signature.getInstance("SHA256withRSA");
    signature.initVerify(publicKey);
    signature.update(data.getBytes(StandardCharsets.UTF_8));
    
    return signature.verify(signatureBytes);
}
```
